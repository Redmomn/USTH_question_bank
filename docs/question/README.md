## 这里面没有我想要的题库

- 请提供题库相关信息给我们，例如课程编码和课程名称
- 或者向我们反馈

1. 首先你需要一个[Gitee账号](https://gitee.com/signup)，建议绑定邮箱，以便题库更新之后的通知
2. 然后来[这里](https://gitee.com/Redmonm/question-bank/issues/new/choose)，选择好合适的模板之后，点击`立即开始`，按照提示填写相应信息
3. 点击`创建`提交issue
4. 等着题库更新
